# 🍔 FoodApp – Responsive Food Ordering UI

## 🚀 Live Demo
https://ashu-food-app.web.app

A modern and responsive Food Ordering UI built using HTML, Tailwind CSS, and Bootstrap.

## ✨ Features
- Responsive design
- Food cards UI
- Carousel offers
- Sticky navbar

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS
- Bootstrap 5
